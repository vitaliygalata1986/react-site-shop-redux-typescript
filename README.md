# Интернет-магазин

Интернет-магазин разработан с использованием такого стека технологий:

1. **React**
2. **TypeScript**
3. **React Redux Toolkit**

### Используемый API

- https://purpleschool.ru/pizza-api-demo

### Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите приложение:
   npm run dev
